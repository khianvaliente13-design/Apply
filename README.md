# 🎬 Khian Personal Portfolio

> Cinematographer · Video Editor · Frontend Developer · Creator of [@catmemeshorts](https://www.facebook.com/profile.php?id=61572818214002)

A dark-themed, animated personal portfolio website built with pure **HTML, CSS, and Vanilla JavaScript** no frameworks, no dependencies.

---

## ✨ Live Features

- ⚡ **Custom cursor** with smooth follower animation and hover scaling
- 🎞️ **Scroll-triggered reveal animations** (fade up, left, right) via Intersection Observer
- 🔢 **Animated stat counters** that count up on scroll
- 🗂️ **Tabbed work section** (Video Editing / Web Projects)
- 🌊 **Parallax hero background text**
- 🧲 **Magnetic button effect** on CTAs
- 🃏 **3D tilt effect** on skill cards
- 📜 **Marquee scrolling ticker** of skills
- 🟢 **Navbar scroll state** + active section highlighting
- 🎉 **Page loader bar** animation on load
- 🐛 Console Easter Egg for devs who peek at the source

---

## 🛠️ Tech Stack

| Layer | Tech |
|---|---|
| Structure | HTML5 |
| Styling | CSS3 (Custom Properties, Flexbox, Grid) |
| Animations | CSS Keyframes + Intersection Observer API |
| Interactivity | Vanilla JavaScript (ES6+) |
| Fonts | Google Fonts — Bebas Neue, DM Sans, Space Mono |

## 🚀 Getting Started

No build tools needed. Just clone and open.

```bash
git clone https://github.com/your-username/portfolio.git
cd portfolio
# Open index.html in your browser
```

Or use a local dev server for best results:

```bash
# With VS Code Live Server extension, or:
npx serve .
```
